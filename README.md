# Raven Security & Surveillance Systems Website

A modern, responsive website for Raven Security and Surveillance Systems - complete home security solutions.

## 📋 Overview

This website showcases comprehensive security solutions for homeowners, including:
- CCTV Surveillance Systems
- Gate Automation
- Intruder Alarm Systems
- Electric Fencing
- Razor Wire Installation
- Car Ports

## 🎨 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Service Showcase**: Detailed information about all security services
- **Contact Integration**: Easy-to-use contact form and direct communication options
- **Accessibility**: Semantic HTML and ARIA labels for better accessibility
- **Performance**: Optimized for fast loading and smooth interactions

## 🚀 Quick Start

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize contact information as needed
4. Add company logo as `logo.png` in the root directory

## 🛠️ Customization

### Update Contact Information
Edit the contact section in `index.html`:
```html
<p><a href="tel:+256704077536">+256 704 077 536</a></p>
<p><a href="mailto:kanjuba@gmail.com">kanjuba@gmail.com</a></p>
<p>Universal House, Kampala, Uganda</p>
```

### Change Colors
Update the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d32f2f;    /* Red */
    --secondary-color: #1a1a1a;  /* Black */
    --text-color: #333;
    --light-bg: #f5f5f5;
    --white: #ffffff;
    --accent-color: #ff6b6b;
}
```

### Add Logo
Replace `logo.png` in the root directory with your company logo.
